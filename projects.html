My projects

A list of my personal and professional projects

MixiPHP (Now Deprecated)

One of my first open-source projects (MIT license), MixiPHP, is a web development framework with an MVC architecture inspired by Laravel. The idea behind this framework was to learn how to develop an MVC architecture framework. Other important concepts I learned while developing it include HTTP methods, defining endpoint routes for REST API-based development, and developing a basic ORM for handling persistent data in the database.

Of course, the development and maintenance of this project can be continued by anyone interested. More technical details about this project can be found in the official GitHub repository.

KoƒçkaPHP

KockaPHP is a personal project inspired by Express JS and, in a way, a continuation of the MixiPHP project. Moving away from the concept of creating a framework for learning and developing under a more modern standard (JavaScript style), the KockaPHP framework was born with the idea of developing REST APIs in an agile and scalable manner using PHP. Although it is still in a very early development stage, I am including this project here because I plan to maintain it for at least a couple of years, depending on the community's response.

This is a project heavily influenced by Express JS and aims to be used to create APIs and to be used with technologies like TailwindCSS and HTMX.

As the development of this project continues, I will post information about its progress on my social media channels.

See Github Repository

Cozy Game

Sprites By Ellenora

As a video game enthusiast, one of my favorite hobbies is game development. This project is a cozy RPG inspired by games like Stardew Valley and Animal Crossing. It is developed entirely in the Godot game engine and is intended as a collaborative project with Ellenora. This simple project serves as both practice for me and a step towards fulfilling one of my dreams: to officially publish a game. Through this project, I aim to refine my skills in game development, work on collaborative creativity, and bring an enjoyable gaming experience to others.

This project is being developed with open-source tools and released in parts and in collaboration, so its source code is reserved. However, the game assets can be found on my official page on Itch.io
